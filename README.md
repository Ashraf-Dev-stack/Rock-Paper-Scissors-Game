# ✊ Rock ✋ Paper ✌️ Scissors Game

Simple 2-player Rock Paper Scissors game built with Python. Challenge your friend and see who's the real winner!

## 🚀 How to Play

1. Clone the repo:
   `bash
   git clone https://github.com/Ashraf-Dev-stack/Rock-Paper-Scissors.git
   
2. *Run the game:*
   python rock_paper_scissors.py
3. *Player 1 picks first:* Type `rock`, `paper`, or `scissors`
4. *Player 2 picks:* Type your move
5. *Winner is announced* and score is updated

🎯 Features
- ✅ *2 Player Mode* - Play with your friend on the same device
- ✅ *Score Tracking* - Automatically counts wins for both players
- ✅ *Input Validation* - Rejects invalid moves, only accepts rock/paper/scissors
- ✅ *Clear Winner Logic* - Rock beats Scissors, Scissors beats Paper, Paper beats Rock
- ✅ *Play Again Option* - Keep playing multiple rounds without restarting
- ✅ *Draw Detection* - Handles ties correctly

🛠️ Built With
- *Python 3* - Core game logic, no external libraries needed
